# Day 25 - Networking with URLSession

## 📖 Giới thiệu
URLSession là API networking chính của iOS. Kết hợp với async/await, việc gọi API trở nên đơn giản và clean.

## 🎯 Kiến thức chính
- URLSession.shared.data(from:)
- async/await networking
- Error handling cho network calls
- URLRequest configuration (headers, method, body)

## 📝 Tóm tắt
- URLSession + async/await = modern networking
- Luôn handle errors và check HTTP status
- Decode JSON response với Codable
- Dùng URLRequest cho custom requests (POST, headers)
