# Day 28 - Combine Framework

## 📖 Giới thiệu
Combine là reactive programming framework của Apple. Publishers emit values over time, Subscribers receive them. Operators transform data streams.

## 🎯 Kiến thức chính
- Publisher & Subscriber
- Common operators: map, filter, debounce, combineLatest
- @Published property wrapper
- Cancellables
- Integration với URLSession

## 📝 Tóm tắt
- Combine = reactive streams cho Swift
- Publishers emit, Operators transform, Subscribers receive
- @Published + ObservableObject cho SwiftUI binding
- AnyCancellable quản lý subscription lifecycle
