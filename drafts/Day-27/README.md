# Day 27 - Core Data Basics

## 📖 Giới thiệu
Core Data là framework persistence của Apple. Nó quản lý object graph và lưu trữ data locally trên device.

## 🎯 Kiến thức chính
- NSManagedObjectContext
- Entity & Attributes
- CRUD operations (Create, Read, Update, Delete)
- NSFetchRequest & predicates
- @FetchRequest trong SwiftUI

## 📝 Tóm tắt
- Core Data = ORM cho iOS
- Context quản lý objects trong memory
- FetchRequest query data từ store
- SwiftUI integration với @FetchRequest
