# Day 26 - JSON Parsing & Codable

## 📖 Giới thiệu
Codable (Encodable + Decodable) là protocol để serialize/deserialize data. Đây là cách standard để parse JSON trong Swift.

## 🎯 Kiến thức chính
- `Codable` protocol
- `JSONDecoder` / `JSONEncoder`
- CodingKeys cho custom mapping
- Nested objects & arrays
- Date decoding strategies

## 📝 Tóm tắt
- Conform to Codable cho automatic JSON parsing
- CodingKeys map JSON keys to Swift properties
- Handle optional fields gracefully
- Custom decoding cho complex structures
