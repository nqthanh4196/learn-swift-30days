# Day 29 - Animations & Gestures

## 📖 Giới thiệu
SwiftUI cung cấp animation system mạnh mẽ và gesture recognizers dễ sử dụng. Kết hợp cả hai tạo ra UI interactive và delightful.

## 🎯 Kiến thức chính
- `.animation()` modifier
- `withAnimation { }` explicit animation
- Spring, easeIn, easeOut, linear
- DragGesture, TapGesture, LongPressGesture
- Transitions (.slide, .opacity, .scale)

## 📝 Tóm tắt
- Implicit animation: attach .animation() to view
- Explicit animation: wrap state change in withAnimation
- Gestures: .gesture() modifier
- Combine gestures with state for interactive UIs
