# Day 28 - WiFi: Web Server trên Pico W

## 📖 Giới thiệu
Tạo HTTP web server trên Pico W. Truy cập từ browser để xem sensor data và điều khiển GPIO.

## 🎯 Kiến thức chính
- TCP server lắng nghe port 80
- Parse HTTP request (GET/POST)
- Serve HTML response
- REST API cho IoT control

## 🏋️ Challenge
1. Serve HTML page hiển thị temperature
2. Button trên web page bật/tắt LED
3. Auto-refresh data mỗi 5 giây
